# Markdown Editor #

This is going to be a fully featured editor for markdown.

## Features ##

- Text edit
- Preview pane
- Save as/New/Open/Close
- Multiple documents via tabbed bar
- Menu bar for quick save/open/new/etc.
- Configuration
- Export html
- Scroll sync with preview (by % not inspecting markdown)
